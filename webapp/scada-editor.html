üìñ How to Use Parameter Mapping

Since the draw.io core files need to be downloaded separately, here's the quick workflow:

Method 1: Use draw.io Online + Our Utility

  1. Create your diagram at app.diagrams.net
  2. For each shape you want to map:
    • Right-click ‚Üí Edit Data
    • Add property: scada-param = TANK_01_LEVEL
    • Add property: scada-type = gradient-level
    • Add property: scada-min = 0
    • Add property: scada-max = 100
  3. Export as SVG
  4. Use our React component to load and update the SVG

Method 2: Manual SVG Editing

  1. Create diagram in any tool and export as SVG
  2. Open SVG in text editor
  3. Add data attributes to elements:
    <rect data-scada-param="TANK_01_LEVEL" data-scada-type="gradient-level" data-scada-direction="bottom-to-top" data-scada-min="0" data-scada-max="100" />
  4. Use our svg-parameter-updater.js in your React app
